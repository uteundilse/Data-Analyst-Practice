5. **星巴克促销活动分析 Python**

  **项目背景**:分析星巴克活动期间不同营销活动对客户交易数据的影响,以确定不同客户群体的定位,提高客户粘性和品牌认知,从而增加总体收入。

  **数据探索与建模**:利用相关系数矩阵和回归分析发现会员收入、活动推送途径及活动类型等影响客户消费。K-means聚类将客户分五组,分析不同特征和响应。

  **项目结果**:发现影响客户消费和品牌忠诚的关键因素,为制定针对性营销计划提供决策支持。
